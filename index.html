Introduction

C++ is a general-purpose compiled programming language. It is generally used in applications where speed, efficiency and thorough control of computer resources is needed, this means that it is suitable for applications such as: operating systems development, video game development, embedded systems development, among others. This ability to work closely with the hardware is why it is called low-level programming language.

C++ is an extension of the programming language called C, adding new features such as the ability to structure programs through an object-oriented paradigm. It is an extension of the programming language called C, adding new features such as the ability to structure programs using an object-oriented paradigm. It also brought with it new things like:

  • STL - Standard Template Library: It provides template classes for commonly used data structures and components for implementing added, built-in functionality. One such component is containers such as Vectors, that store collections of objects.
  • Namespaces: They are containers used to organize code into logical groups of identifiers and similar objects under a name, within a scope. They prevent name colisions when multiple libraries are present, and prevent conflict with names of other namespaces within a program.
  • Exception handing: C++ supports exception handling by introducing try and catch blocks.

All these new options make software written in C++ more scalable and maintainable without losing speed and efficiency. The information of this section was get from this FreeCodeCamp entry.

Getting started

Macros

Macros are directives for the preprocessor. They allow the programmer to work with different libraries, define constants, define reduced statements, create custom functions, control how the program is compiled according to certain software and/or hardware characteristics, define which version of C++ the script will be compiled with, among many other things.

One of the simplest but most important macros will be presented in this documentation. This macro is called #include which will allow us to include libraries to use different functions, such as reading data entered from the keyboard, as well as displaying this data through the system console. To use the library with the functions described above, the following code is used:

#include <iostream>

You can read the following documentation about C++ standard libraries for more information on all the existing libraries for this programming language.

Namespace std

As described above, namespaces are containers used to organize code into logical groups of identifiers and similar objects under a name. One of the most used namespaces is std, in it are grouped a great variety of functions, among them those of reading data and printing data on the screen. Its syntax is as follows:

using namespace std

To use it, you only have to invoke the name of the function you want to access. For example, if you want to print a text on the screen, you would use the following statement:

cout << Hello world!;

If the use of the workspace were not defined, every time you want to use a function belonging to it, you would have to use the following statement:

std :: cout << Hello world!;

Therefore, it is always advisable to define the use of the namespace at the beginning of the program.

Main function

The main function is the place where the program is going to start executing, it must always be present in any program written in C++. Its syntax is as follows:

int main()
{

   return 0;
}

You can see how at the end of the main function there is a return 0, it is a directive of the functions that we are not going to see in this documentation but basically tells the compiler that the program was executed correctly (only in the case of the main function).

Comments

Comments are lines that are not taken into account when compiling the program, i.e. they have no effect on the execution of the software. This is a great help because you can leave text in the program to describe the operation of parts of the code, explain what certain variables and functions are used for, among other things. Comments are mainly divided into two types: single-line and multi-line. The syntax is as follows:

// This is a single-line commentary

/* This is a
   multi-line
   commentary */

Example

Having seen the main components of a C++ program, we will show an example of a program that prints the phrase "Hello world!". The code is as follows:

#include <iostream>

using namespace std

// This program print a text on the console
int main()
{
   cout << Hello world!;
   return 0;
}

Data types and variables

C++ has a large number of data types for classify information, this information is stored in a variable. A variable has a value and a data type associated with it. The most common ones and ther form of use are shown below:

  • bool: A data type to represent true or false.
    bool led = true;
  • char: A data type asocciated with characters. char:
    char letter = 'a';
  • double: A data type asocciated with large decimal numbers.
    double PI = 3.14159265358979323846264;
  • float: A data type asocciated with decimal numbers.
    float PI = 3.141592653;
  • int: A data type asocciated with integers numbers.
    int humanity_age = 350000;

These data types have a maximum capacity of information they can store. This capacity is measured in bytes (one byte is 8 bits). Below is the maximum value that can be stored in each of the data types seen.

  • bool: (1 byte) = -128 to +128
  • char: (1 byte) = 0 to 255
  • double: (8 bytes) = 1.7e-308 to 1.7e+308
  • float: (4 bytes) = 3.4e-38 to 3.4e+38
  • int: (4 bytes) = -2147483648 to +2147483648

Depending on the application to be given to the variables, the appropriate data type must be chosen, taking into account both the size of the information to be handled and the required precision.

Example

A single example of declarations and asignations of variables.

#include <iostream>

using namespace std

int main()
{
   int age;
   age = 24;

   float grade;
   grade = 5.0;

   return 0;
}

Decisions statements

Decision statements are parts of the program that, given certain conditions, allow the program to execute a given section. The following are the most common decision statements.

  • if (condition): This statement executes a block of code if the condition inside the parentheses is true.
    if (condition)
    {
       // Code block to execute
    }
  • else: This statement must go together with an if statement, when the condition of the if statement is not true, the code block assigned to the else statement is executed.
    if (condition)
    {
       // Code block to execute
    }
    else
    {
       // Code block to execute when condition of if is false
    }
  • else if (condition): This statement works similar to the else statement, it is executed when the condition of the if statement is not true, but it also has its own condition to be executed. This means that it requires two conditions to be executed, an if with a false condition and the else if condition is true.
    if (condition)
    {
       // Code block to execute
    }
    else if (condition)
    {
       /* Code block to execute when condition of if is false
       and the condition of the else if is true */
    }

Example

A single example to use the conditional statements.

#include <iostream>

using namespace std

int main()
{
   if (condition)
   {
      // Code block to execute
   }
   else if (condition)
   {
      /* Code block to execute when condition of if is false
      and the condition of the else if is true */
   }
   else
   {
      // Code block to execute when a condition of any if is false
   }

   return 0;
}

Loop statement

Loop statements allow a block of code to be executed a certain number of times, either until a certain condition is met or to be executed indefinitely. The following are the one most common loop statement.

  • while (condition): This statement executes a code continuously until the condition inside the parentheses is false.
    while (condition)
    {
       // Code block to execute in loop
    }

Example

A single example to use the loop statement.

#include <iostream>

using namespace std

int main()
{
   while (condition)
   {
      // Code block to execute in loop
   }

   return 0;
}
